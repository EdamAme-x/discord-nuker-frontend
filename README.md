# discord-nuker-frontend
Discord荒らしツールのフロントエンドです。 Sponsor: 荒らし共栄圏

### Setup

```shell
pnpm i
pnpm dev
```

### Shadcn add UI
```shell
pnpm add-ui {UINAME}
```

### Formt & Lint
```shell
pnpm lint
pnpm format
```

### Start Server
```shell
pnpm dev
```

## Stacks
- Next.js
  - AppRouter
  - RSC
- ShadcnUI
  - TailwindCSS
  - useHooks
